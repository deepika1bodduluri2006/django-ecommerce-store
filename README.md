# 🛒 Django E-Commerce Store

A modern E-Commerce web application built using **Python, Django, SQLite, HTML, CSS, and Bootstrap**. This project allows users to browse products, register/login, add products to a shopping cart, place orders, and view their order history.

---

## 📌 Features

### 👤 User Authentication
- User Registration
- User Login
- User Logout
- Secure Authentication using Django

### 🛍️ Product Management
- View all products
- Product categories
- Product details page
- Product images
- Product descriptions

### 🛒 Shopping Cart
- Add products to cart
- Increase quantity
- Decrease quantity
- Remove products
- View total price

### 📦 Order Management
- Checkout page
- Place orders
- Order history
- Order details

### 🎨 User Interface
- Responsive Bootstrap design
- Navigation bar
- Category buttons
- Clean and simple layout

# 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/deepika1bodduluri2006/django-ecommerce-store.git
```

### Go to project folder

```bash
cd django-ecommerce-store
```

### Install Django

```bash
pip install django
```

### Run migrations

```bash
python manage.py migrate
```

### Start the server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
